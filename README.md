# 🌾 KISAN SAATHI PRO

## Your AI-Powered Smart Farming Assistant

---

## About

**Kisan Saathi Pro** is a desktop application for Indian farmers with AI-powered crop recommendations, real-time weather, disease detection, profit calculator, farm diary, and bilingual support (English/Hindi).

---

## Features

| Feature | Description |
|---------|-------------|
| 🌾 Crop Advisor | Crop recommendations based on soil type |
| ☀️ Live Weather | Real-time weather with farming tips |
| 📍 GPS Location | Auto-detect your location |
| 🦠 Disease Detection | Identify diseases from symptoms |
| 💰 Profit Calculator | Calculate profit per acre |
| 📖 Farm Diary | Track farming activities |
| 🤖 AI Assistant | Ask farming questions |
| 🌐 Bilingual | English / Hindi |

---

## Download

### Option 1: EXE File (Easy)
Download `KisanSaathiPro.exe` from Releases folder and double-click to run.

### Option 2: Run from Code

```bash
git clone https://github.com/ssharmar43-collab/kisan-saathi-pro.git
cd kisan-saathi-pro
pip install customtkinter requests geocoder
python app.py
```

---

## How to Use

1. **Register** - Create account with your farm details
2. **Login** - Use email and password
3. **Dashboard** - View your crops and weather
4. **Crop Advisor** - Find best crops for your soil
5. **Weather** - Check real-time weather
6. **Disease Detection** - Describe symptoms, get treatment
7. **Profit Calculator** - Calculate expected profit
8. **Farm Diary** - Record daily activities
9. **AI Assistant** - Ask farming questions

---

## Tech Stack

- Python 3.11+
- CustomTkinter (UI)
- SQLite3 (Database)
- Open-Meteo API (Weather)
- Geocoder (GPS)
- PyInstaller (EXE)
